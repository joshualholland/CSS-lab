Cyberpunk 2077

Cyberpunk 2077 is a new game made by CD Projekt, the same studio that made The Witcher 3. It is going to be an open world RPG set in the future. You play as V, a "cyberninja" with all sorts of special skills and abilities. Think of this game as sort of a GTA meets witcher. You can do everything from drive cars, wield insane guns, and have romantic relationships. The estimated release year is currently 2019, with some speculation of possibly even 2020.
